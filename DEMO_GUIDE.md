# 🎯 CampusKart Demo Guide

## Quick Demo Overview

This guide will help you showcase all features of CampusKart effectively.

## 🔐 Demo Accounts

For the best demo experience, you can use these approaches:

### Option 1: Create a New Account
1. Go to the landing page
2. Click "Create Account"
3. Fill in: Name, College, Email, Password
4. You'll have access to all user features immediately

### Option 2: Use Sample Data
The app is pre-populated with sample data including:
- 8 products across various categories
- 3 flat listings
- Sample chat conversations
- Wishlist items
